# Poems_Writing
项目全名为“古诗词基础学习与鉴赏”，里面一共分为三个页面，在第一页用户可以看到各种各样的写诗写词的技巧方法干货，如诗的种类、押韵、抒情方式、意象搭配、意境营造、表达技 巧等等；在第二页我们推荐了一些名人的优秀而简单的诗作；第三页是一个拓展学习，诗中有画，画里有诗，在欣赏的同时用户也可能获得自己创造诗歌的灵感 
功能：
1.知识干货
2.好诗推荐
3.画里有诗
